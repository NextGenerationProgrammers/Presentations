Spaltenorientierte Datenbanken

Abfrage wie bei Zeilenorientierten Datenbanken

Zeilenorientiert

MoeMary11.11.1942

Spaltenorientiert

DoeMoe
JohnMary
12.07.196511.11.1942

Vorteile

  • Auswertung von gleichen Daten performanter
  • Statistiken können schnell dargestellt werden
  • Durchschnitte können schneller berechnet werden
Nicht alle Datensätze müssen Stück für Stück gelesen werden, sonder nur einer

Bekannte Vertreter

  • HBase von Apache

Objektorientierte Datenbanken

  • Speichert Daten analog zur OOP
  • Keine Fremdschlüssel

Bekannte Vertreter

  • db4o
db4o - Database for Objects

Key-Value Stores

  • Vergleichbar mit sehr großer HashMap (Java)/Dictionary (.NET)
  • Speicherung als Schlüssel-Wert Paare
  • Value ist immer nur ein Wert (aber evtl. JSON)
  • Verwendung als Zwischenspeicher (im RAM - nicht als persistenter Speicher)

Vorteile

  • Schnell
  • Einfach aufgebaut

Bekannte Vertreter

  • Riak
  • Redis

Dokumenten-Datenbanken

  • Daten werden als Dokumente gespeichert
  • Nicht normalisiert
  • z.B. JSON Format

Vorteile

  • Keine Fremdschlüssel
  • Kein festes Schema
  • In JS einfach zu verarbeiten

Bekannte Vertreter

  • Mongo DB
  • Couch DB
