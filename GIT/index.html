GIT Allgemein
Git ist eine freie Software zur verteilten Versionsverwaltung von Dateien
Die Entwickler sind Junio C Hamano, Shawn O. Pearce und Linus Torvalds.
Die aktuellste Version ist 2.19.1
Was macht Git
Snapshots, nicht Diffs
3 Schritte bei Git
Du Bearbeitest Dateien in deinen Arbeitsverzeichnis Du makierst DAtien für den nächsten Commit, indem Du Snapshots zur Staging Area Hinzufügst Du legst den Commit an, wodurch die in der Staging Area vorgemerkten Snapshots dauerhaft im Git Verzeichnis gespeicher werden
Eigenschaften von GIT
·Nicht-linear Entwickelt
·Kein Zentraler Server
·Datentranfer zwischen Repostorien
·Kryptographische Sicherheit der Projektgschichte
·Sichersystem und Datenversionierung
·Säubern des Repositories
·Interoperabilität
·Web-Interface
Verwendung
Die aktuelle Version wird produktiv für die Entwicklung vieler Open-Source-Projekte eingesetzt. Außerdem wird Git von einer Vielzahl Weiterer Open-source-Projekte genutzt.
