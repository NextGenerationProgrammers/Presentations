Polymorphie, Überladen, Überschreiben

Was sind überladene Methoden?

Methoden mit dem selben Namen,

die sich in der Parameteranzahl

oder in den Parametertyp(en)

und in der Implementierung

unterscheiden

class A
{
    public void Foo()
    {
        Console.WriteLine("Hello from Foo!");
    }
    public void Foo(string text)
    {
        Console.WriteLine($"Hello from Foo with {text}");
    }
}
class A
{
    public int Add(int a, int b)
    {
        return a + b;
    }
    public double Add(double a, double b)
    {
        return a + b;
    }
}

Was ist Überschreiben?

bei geerbten Methoden kann die Implementierung verändert werden, wenn es die Grundklasse erlaubt.

In C# werden hierfür die Wörter virtual, abstract und override verwendet

class Base
{
    public virtual void Foo()
    {
        Console.WriteLine("Hello Base!");
    }
}
class Derived : Base
{
    public override void Foo()
    {
        Console.WriteLine("Hello Derived!");
    }
}

Was ist Polymorphie?

Variablen können zur Laufzeit verschiedene Typen annehmen und ihr Verhalten dementsprechend verändern.

abstract class Base
{
    public abstract void Foo();
}
class DerivedA : Base
{
    public override void  Foo()
    {
        Console.WriteLine("Hello A!");
    }
}
class DerivedB: Base
{
    public override void Foo()
    {
        Console.WriteLine("Hello B!");
    }
}

Welche Ausgabe hat folgendes Programm?

static void Main(string[] args)
{
    Base val = new DerivedA();
    val.Foo();
    val = new DerivedB();
    val.Foo();
    Console.ReadLine();
}
